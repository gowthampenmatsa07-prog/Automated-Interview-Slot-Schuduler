# Automated-Interview-Slot-Schuduler
Automated Interview Slot Scheduler is a system that automates interview scheduling by matching candidate and interviewer availability. It reduces manual effort, avoids scheduling conflicts, and sends automatic notifications, making the recruitment process faster, more efficient, and user-friendly.
Overview

The Automated Interview Slot Scheduler is a web application that simplifies the interview scheduling process by automatically matching candidate availability with interviewer schedules. It reduces manual effort, minimizes scheduling conflicts, and streamlines communication through automated notifications.

Features
Candidate interview slot booking
Interviewer availability management
Automatic schedule matching
Conflict-free slot allocation
Email/notification reminders
Admin dashboard for monitoring schedules
User-friendly interface
Technologies Used
Frontend: HTML, CSS, JavaScript (or React)
Backend: Node.js / Python / Java / PHP
Database: MySQL / PostgreSQL / MongoDB
Notification Service: Email API (SMTP, SendGrid, etc.)
System Workflow
Interviewers provide available time slots.
Candidates select preferred interview times.
The system automatically assigns available slots.
Confirmation notifications are sent to both parties.
Reminders are generated before the interview.
